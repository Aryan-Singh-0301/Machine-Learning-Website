Back-ground Informaton on Machine Learning

Machine Learning is a type of Artificial Creation, via humans, of human intelligence. The concept of machine Learning implies a thought-process where a machine can learn, reason, plan, percieve or process natural Language. These concepts allow any AI to bring many socio-economic advantages, while also bringing in many ethical issues.


Machine Learning is seen as a subset of Artificial intelligence, and contains algorithms that help to build a mathematical model for some sample data. The machine then uses the given data and the algorithm to make predictions/decisions about other event without explicitly being told do so. Machine Learning is very closely related to computational statistics, in which, a machine proceeds about making a prediction from a particular given set of data.



History of Machine Learning

The term Machine Learning was first recognized by a person named Author Samuel in 1959. He was an American IBMer in the field of computer Gaming and Artificial intelligence. A book developed for particularly for Machine Learning at that time were the Nilssons book on Machine Learning. These books mainly dealt with the concepts of patterns of classification. These new concepts gave rise to new interests in individuals, which continued into the 1970's. During 1981 a report was published, describing on how to use such patterns and strategies so that a neural network learns to recognize 40 characters, which are comprimised of: 26 letters, 10 digits, 4 special symbol.


Later on, Tom M. Mitchell provided a more generalized quote for the algorithms comprimised in Machine Learning. He quoted, "A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P if its performance at tasks in T, as measured by P, improves with experience E." This new definition of the algorithms in Machine Learning helped to further grasp the core understanding of the algotithms involved in Machine Learning, rather than simply understanding it in cognitive terms. Following this definition was Alan Turings propsal paper known as "Computing Machinery And Intelligence". In this article the questions that "Can Machines Think?" was modified by the question "Can machines do what we can do?".This question further lead into the understanding of Machine Learning as it is now.
